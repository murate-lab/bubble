# Bubble Game

Webカメラの映像上にシャボン玉を浮かばせ、体の動きで割るゲームです。

## デモ

シャボン玉が画面下から浮かび上がり、体を動かすと割れます。サイズに応じた効果音が鳴り、スコアが加算されます。

## 動作環境

- Windows 10/11 (x64)
- Webカメラ (デフォルト: 1280x720)
- Visual Studio 2019 以降

## 依存ライブラリ

- [OpenCV](https://opencv.org/) 4.x 以降

## ビルド方法

1. OpenCV をインストールし、インクルードパスとライブラリパスを Visual Studio のプロパティで設定する
2. `bubble.sln` を Visual Studio で開く
3. 構成を `Release / x64` に設定してビルドする

## 使い方

```
bubble.exe [width] [height] [bubble_size_max]
```

| 引数 | 説明 | デフォルト |
|------|------|-----------|
| `width` | カメラ映像の横幅 (px) | 1280 |
| `height` | カメラ映像の縦幅 (px) | 720 |
| `bubble_size_max` | シャボン玉の最大サイズ (px) | 400 |

- `Esc` キーで終了
- 実行するとタイムスタンプ付きの MP4 ファイルが自動生成されます

## ゲームの仕組み

- 3フレーム差分を使って体の動きを検出します
- シャボン玉はランダムなサイズ (3段階) と横揺れで画面を浮かび上がります
- シャボン玉の領域内で動きが検出されると、サイズに応じた効果音を再生して消去しスコアを加算します

## ファイル構成

```
bubble/
├── bubble.cpp        # メインロジック
├── WebCam.cpp/h      # Webカメラ取得スレッド
├── bubble.png        # シャボン玉画像 (RGBA)
├── bubble1.wav       # 効果音 (小)
├── bubble2.wav       # 効果音 (中)
└── bubble3.wav       # 効果音 (大)
```
